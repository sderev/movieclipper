# Documentation Guide

Welcome to the Movie Clipper documentation! This guide will help you navigate the documentation in the most effective order based on your needs and experience level.

## Quick Start Path (New Users)

**Goal**: Get your first clip created in 5 minutes

1. **[Installation Guide](INSTALLATION.md)** - Set up the tool
2. **[Usage Guide](USAGE.md)** - Create your first clips
3. **[Audio Guide](AUDIO.md)** - Understand DaVinci Resolve compatibility

**Stop here if**: You just want to create basic clips for video editing.

## Complete User Path

**Goal**: Master all features and workflows

1. **[Installation Guide](INSTALLATION.md)** - Complete setup
2. **[Usage Guide](USAGE.md)** - Common patterns and examples
3. **[Audio Guide](AUDIO.md)** - Audio handling mastery
4. **[Advanced Guide](ADVANCED.md)** - Performance and power features
5. **[Troubleshooting](TROUBLESHOOTING.md)** - Problem solving
6. **[Reference](REFERENCE.md)** - Complete command reference

## Problem-Solving Path

**Goal**: Fix specific issues or learn about specific features

- **Having installation issues?** → [Installation Guide](INSTALLATION.md)
- **Movie not found or audio problems?** → [Troubleshooting](TROUBLESHOOTING.md)
- **DaVinci Resolve compatibility issues?** → [Audio Guide](AUDIO.md)
- **Want better performance?** → [Advanced Guide](ADVANCED.md)
- **Need exact command syntax?** → [Reference](REFERENCE.md)

## Documentation Files Overview

### [Installation Guide](INSTALLATION.md)
**When to read**: First time setup or reinstallation  
**Time needed**: 10-15 minutes  
**What you'll learn**: Prerequisites, installation, first-time configuration, directory setup

### [Usage Guide](USAGE.md)
**When to read**: After installation, for daily usage  
**Time needed**: 15-20 minutes  
**What you'll learn**: Basic commands, fuzzy matching, time formats, common workflows

### [Audio Guide](AUDIO.md)
**When to read**: When working with DaVinci Resolve or multiple languages  
**Time needed**: 20-25 minutes  
**What you'll learn**: Audio stream selection, DaVinci Resolve compatibility, language handling

### [Advanced Guide](ADVANCED.md)
**When to read**: When you want to optimize performance or use power features  
**Time needed**: 25-30 minutes  
**What you'll learn**: Caching system, symlinks, batch processing, configuration tuning

### [Troubleshooting](TROUBLESHOOTING.md)
**When to read**: When you encounter problems  
**Time needed**: 5-10 minutes per issue  
**What you'll learn**: Common problems and solutions, debugging techniques

### [Reference](REFERENCE.md)
**When to read**: When you need exact command syntax or configuration details  
**Time needed**: As needed (reference document)  
**What you'll learn**: Complete CLI reference, configuration options, technical details
